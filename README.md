# Movie Recommendation System

A Movie Recommendation System built using Python and Machine Learning.<br>
This project recommends movies based on similarity using content-based filtering.<br><br>

It analyzes movie genres and keywords to suggest similar movies to the user.

# Features

✔ Recommend movies based on similarity<br>
✔ Uses content-based filtering<br>
✔ Case-insensitive movie search<br>
✔ Clean and modular code structure<br>
✔ Interactive CLI interface


## Concepts Used

• Content-Based Filtering<br>
• Text Vectorization (CountVectorizer)<br>
• Cosine Similarity<br>
• Natural Language Processing (NLP)

##  Tech Stack

Python 3<br>
Pandas<br>
Scikit-learn
